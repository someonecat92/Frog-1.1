# Frog — объектно‑ориентированный язык программирования

[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE.md)
[![Java](https://img.shields.io/badge/language-Java-orange.svg)](https://www.java.com/)
[![Status: Experimental](https://img.shields.io/badge/status-experimental-yellow.svg)]

Frog — экспериментальный объектно‑ориентированный язык с минималистичным синтаксисом. Язык спроектирован с упором на чёткую модульную архитектуру и расширяемость: все компоненты (лексер, парсер, AST, интерпретатор) строго разделены по пакетам и файлам. Frog поддерживает классы, поля, базовые операторы и простые выражения, позволяя быстро прототипировать логику и строить собственные расширения.

![Frog Logo](assets/frog-logo.png)

## Ключевые особенности

- **Объектно‑ориентированная модель**: программы строятся вокруг классов (`class`), полей и операторов, выполняемых в контексте класса.
- **Минималистичный декларативный синтаксис**: поля задаются как `key = ["value"]`, операторы — в понятной форме (`println`, `calc`, `import`, `return`).
- **Поддержка выражений**: арифметические операции (`2+2`, `10*5`) и работа с переменными.
- **Модульная архитектура**: лексер, парсер, AST и интерпретатор строго разделены по пакетам и файлам — это упрощает поддержку, тестирование и расширение языка.
- **Готовность к расширению**: структура проекта позволяет добавлять новые конструкции (условия, циклы, методы) без затрагивания существующих компонентов.

## Структура проекта

Проект организован по принципу «всё раздельно» — каждый компонент находится в собственном файле и пакете:

frog-project/
├── src/
│ └── frog/
│ ├── Main.java # Точка входа: запуск интерпретатора
│ ├── parser/
│ │ ├── Token.java # Типы и экземпляры токенов
│ │ ├── Lexer.java # Лексический анализ
│ │ └── Parser.java # Синтаксический анализ и построение AST
│ ├── ast/
│ │ ├── Node.java # Базовый класс узлов AST
│ │ ├── Program.java # Корневой узел программы
│ │ ├── ClassDef.java # Определение класса
│ │ ├── FieldAssignment.java # Присваивание полей
│ │ ├── Expr.java # Абстрактный узел выражений
│ │ │ ├── NumberLiteral.java
│ │ │ ├── StringLiteral.java
│ │ │ └── BinaryExpr.java
│ │ ├── Statement.java # Базовый класс операторов
│ │ │ ├── PrintStmt.java
│ │ │ ├── ImportStmt.java
│ │ │ ├── CalcStmt.java
│ │ │ └── ReturnStmt.java
│ └── interpreter/
│ ├── Environment.java # Среда выполнения (переменные, контекст)
│ └── Interpreter.java # Выполнение AST
├── assets/
│ └── frog-logo.svg # Логотип языка Frog
├── example.frog # Пример исходного кода на Frog
├── LICENSE.md # Лицензия (MIT)
└── README.md # Эта документация
